\subsection{Detailed Description}
Module reading information of a network adapter. 

Parameters that are observed are\+:
\begin{DoxyItemize}
\item received data (byte, packates)
\item transmitted data (byte, packates)
\item dropped data (transmitted and received)
\item collisions
\end{DoxyItemize}

Hard disk devices are mounted at a certain location. The statfs that is used internally to read information needs the mount point and not the device itself. E.\+g. /dev/sdb1 mounted at /media/data -\/$>$ statfs works on /media/data. Therefore the module might be used multiple times for different mount point to be monitored.

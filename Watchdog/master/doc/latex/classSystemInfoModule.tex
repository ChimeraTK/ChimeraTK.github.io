\subsection{Detailed Description}
This module is used to read system parameter. 

Some of them are static and only read once (e.\+g. processor model). Others are watched continuously (e.\+g. uptime, work load...). The \char`\"{}\+S\+Y\+S\char`\"{} tag is used for all variables that are updated in the main loop. \begin{DoxyRefDesc}{Todo}

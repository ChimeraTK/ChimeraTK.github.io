MotorDriverCard

The MotorDriverCard library provides an interface to the MD22 motor driver card FMC. It offers facilities for low-level communication as well as high-level functionality to operate the motor.

For communication to the motor driver and motor controller of the MD22, SPI commands are handled and sent via PCIe. Hence, the corresponding firmware on the FMC carrier only has to adapt between PCIe and SPI interfaces. For operation of a motor, a state machine representing the basic operation modes and error checking are implemented. All position values can be entered and retrieved in a user-defined unit. The main user interface is provided by the StepperMotor and LinearStepperMotor classes.

\subsection{Detailed Description}
This module is used to read system parameter. 

Some of them are static and only read once (e.\+g. processor model). Others are watched continuously (e.\+g. uptime, work load...). \begin{DoxyRefDesc}{Todo}
\item[\hyperlink{todo__todo000002}{Todo}]Implement proper data types instead of using int for all of them! \end{DoxyRefDesc}

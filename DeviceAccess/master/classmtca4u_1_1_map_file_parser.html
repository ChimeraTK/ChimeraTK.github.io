std::pair< std::string, std::string > mtca4u::MapFileParser::splitStringAtLastDot ( std::string  moduleDotName)
static

Split the string at the last dot.

The part up to the last dot is the first returned argument, the part after the last dot is the second. Hence, the first part can contain dots itself, the second part cannot. If there is no dot, the first part is empty and the full string is returned as second (the part up to the first dot is considered as prefix).

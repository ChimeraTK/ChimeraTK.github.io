Detailed Description

template<class T, class IMPL_T>
class ChimeraTK::TypeChangingStringImplDecorator< T, IMPL_T >

This class is intended as a base class and intentionally does not call shutdown in the destructor in order not to shadow the shutdown detection for the child classes, so it is not forgotten there.

It provides only partial template specialisations for the string stuff. Don't directly instantiate this class. The factory would fail when looping over all implementation types.

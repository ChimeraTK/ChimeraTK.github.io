Detailed Description

template<typename UserType>
class ChimeraTK::ScalarAccessor< UserType >

Accessor for scalar variables (i.e.

single values). Note for users: Use the convenience classes ScalarPollInput, ScalarPushInput, ScalarOutput instead of this class directly.

StepperMotorError ChimeraTK::StepperMotor::getError ( )

return error code for the motor.

The error code is not a bit-field 0 - NO_ERROR 1 - ACTION_ERROR if any of the action started by the user was not successful 2 - MOVE_INTERUPTED if the motion of the motor was interrupted, e.g. a software/hardware limit was hit. 3 - CALIBRATION_LOST if calibration is lost.

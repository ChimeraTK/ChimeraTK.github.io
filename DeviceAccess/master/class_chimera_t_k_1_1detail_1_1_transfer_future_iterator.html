Detailed Description

Internal class needed for TransferElement::readAny(): Provide a wrapper around the list iterator to effectivly allow passing std::list<TransferFuture>::iterator to boost::wait_for_any() which otherwise expects e.g.

std::list<TransferFuture::PlainFutureType>::iterator. This class provides the same interface as an interator of the expected type but adds the function getTransferFuture(), so we can obtain the TransferElement from the return value of boost::wait_for_any().

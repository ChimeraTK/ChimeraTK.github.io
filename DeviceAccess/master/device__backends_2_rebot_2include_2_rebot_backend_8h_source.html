28  // A helper class that contains a mutex and a flag.
29  // The idea is to put it into a shared pointer and hand it to a thread which is sleeping a long
30  // time. You can then detach the thread, tell it to finish and continue with the constructor
31  // without having to wait for the tread to wake up and finish before you can join it.
32  // The thread locks the mutex and checks if it should finish when it wakes up, which it
33  // can do because the mutex and flag still exist thanks to the shared pointer.
34  struct ThreadInformerMutex{
35  std::mutex mutex;
36  bool quitThread;
37  ThreadInformerMutex() : quitThread(false){}
38  };

Detailed Description

Base class for plugins providing additional information in a RegisterInfo, e.g.

coming from the map file. This base class is essentially empty, the derived plugin classes should contain public data members and implement the createInstance() function which is passed to RegisterPluginFactory::registerPlugin(). The data members should be filled by the createInstance() implementation.

Detailed Description

template<typename UserType, typename DataConverterType, bool isRaw>
class ChimeraTK::detail::NumericAddressedPrePostActionsImplementor< UserType, DataConverterType, isRaw >

We need partial template specialisations of some functions.

However, in C++ this is only possible for full classes. Hence we introduce an implementor class which only holds the functions which we have to re-implement anyway. The functions are doPostRead, doPreWrite and doPostWrite.

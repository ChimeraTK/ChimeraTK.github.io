void mtca4u::LinearStepperMotor::emergencyStop ( )
virtual

Emergency stop the motor by disabling driver.

Function stops the motor by disabling driver. It means that it is done in the fastest possible way.
After emergency stop motor calibration can be lost so execution of this method influence calibration status.
Function internally calls 'void stopMotor()' method so all blocking functions will be interrupted.

LinearStepperMotorStatusAndError mtca4u::LinearStepperMotor::getStatusAndError ( )

Return object which contains current status and error.

Returns
LinearStepperMotorStatusAndError - object which contains of status and error code

Return current motor status and error as LinearStepperMotorStatusAndError object.
It give possibility to check what was an error type when status is set to LinearStepperMotorStatusAndError::M_ERROR. Status and error create pair which describes current state of the object. Every time it is called it checks status of the motor.

Additionally to status of StepperMotor extra state types has been added to :
1) StepperMotorStatusTypes::M_POSITIVE_END_SWITCHED_ON - hardware positive end switch has been triggered. Motor is on positive end.
2) StepperMotorStatusTypes::M_NEGATIVE_END_SWITCHED_ON - hardware negative end switch has been triggered. Motor is on negative end.
Additionally to errors of StepperMotor extra error type has been added to :
1) StepperMotorErrorTypes::M_BOTCH_END_SWITCH_ON -both end switch active. Hardware is missing or in error state.

LinearStepperMotorStatusAndError mtca4u::LinearStepperMotor::moveToPosition ( float  newPosition)

Sending motor to new position (blocking).

Parameters
newPosition- new position for the motor
Returns
LinearStepperMotorStatusAndError - status of the motor after movement routine finish

This method sends motor to new position. It is blocking blocking method. It will finish when motor movement will end with success or error. Function can be interrupted by using 'void stopMotor()' method.
Function automatically detects end switches conditions.
It allows to perform movement in case NEGATIVE END SWITCH is active but target position in grater than current one.
It allows to perform movement in case POSITIVE END SWITCH is active but target position in smaller than current one.
Function returns StepperMotorStatus object which is status of the motor after movement routine finished. Should be LinearStepperMotorStatusTypes::M_OK in case of success.

void mtca4u::LinearStepperMotor::stop ( )
virtual

Stop the motor.

Function stop the motor in a way that real position from the motor is set as a target position.
It also interrupts all blocking functions as 'LinearStepperMotorStatusAndError moveToPosition(float newPosition)' or 'StepperMotorCalibrationStatus calibrateMotor()'.
Delays in communication can trigger effect that motor will move a few steps in current movement direction and later will move couple steps back, but this issue can be filtered by motor hysteresis effect.

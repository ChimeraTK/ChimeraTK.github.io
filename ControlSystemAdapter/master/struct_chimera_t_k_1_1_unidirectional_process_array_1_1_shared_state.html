template<class T>
boost::lockfree::spsc_queue< TransferFuture::PlainFutureType > ChimeraTK::UnidirectionalProcessArray< T >::SharedState::_fullBufferQueue

Queue holding the indices of the full buffers.

Those are the buffers that have been sent but not yet received. We can use an spsc_queue for this queue because it is only filled by the sending process array and only consumed by the receiving process array. In fact, we have to use an spsc_queue because the standard boost::lockfree::queue cannot hold futures (as they are not trivially assignable / destructable).

std::atomic<Buffer*> ChimeraTK::UnidirectionalProcessArray< T >::SharedState::_tripleBufferIndex

Special atomic triple buffer to be filled when the queue runs over (i.e.

all normal buffers are full). After construction, buffer 2 is owned by the sender, 3 by the shared state and 4 by the receiver. A buffer is considered containing valid data if its _isValid flag is set. The buffer's default constructor sets _isValid to false.

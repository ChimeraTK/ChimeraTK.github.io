Detailed Description

template<typename UserType>
class ChimeraTK::ArrayAccessor< UserType >

Accessor for array variables (i.e.

vectors). Note for users: Use the convenience classes ArrayPollInput, ArrayPushInput, ArrayOutput instead of this class directly.

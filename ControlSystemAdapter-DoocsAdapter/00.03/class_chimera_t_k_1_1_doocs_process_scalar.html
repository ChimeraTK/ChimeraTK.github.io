Detailed Description

template<typename T, typename DOOCS_T, typename DOOCS_VALUE_T>
class ChimeraTK::DoocsProcessScalar< T, DOOCS_T, DOOCS_VALUE_T >

The DoocsProcessScalar has three template parameters:

  • T, The primitive value type of the ChimeraTK process variable
  • DOOCS_T, The Doocs type which is used
  • DOOCS_VALUE_T, The primitive type of the Doocs type, which is not necessarily the same as T

The last type is necessary for the set_value overloading to work. If for instance T is short, then D_int is used and the signature is set_value(int), not set_value(short).

Detailed Description

Module used to read external log file in order to make messages available to the control system.

E.g. the ChimeraTk watchdog starts a process and its output is not available by the watchdog. But the log file produced by the process can be read. If the log file is set via logFileExternal it is parsed and the tail is published to LogFileTailExternal.

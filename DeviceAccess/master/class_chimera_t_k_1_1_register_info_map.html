Detailed Description

Provides container to store information about registers described in MAP file.

Stores detailed information about all registers described in MAP file. Provides functionality like searching for detailed information about registers and checking for MAP file correctness. Does not perform MAP file parsing

bool ChimeraTK::RegisterInfoMap::check ( ErrorList &  err,
ErrorList::ErrorElem::TYPE  level 
)

Checks logical correctness of MAP file.

Checks if names in MAP file are unique and if register addresses overlappe. Errors are not reported if two registers with the same name have the same parameters. Checks only syntactic correctness of data stored in MAP file. Syntax and lexical analizys are performed by MAP file parser.

Parameters
errlist of detected errors
levellevel of checking - if ERROE is selected only errors will be reported, if WARNING is selected errors and warning will be reported
Returns
false if error or warning was detected, otherwise true

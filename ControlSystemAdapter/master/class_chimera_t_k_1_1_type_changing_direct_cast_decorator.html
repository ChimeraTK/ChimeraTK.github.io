Detailed Description

template<class T, class IMPL_T>
class ChimeraTK::TypeChangingDirectCastDecorator< T, IMPL_T >

This decorator does not do mathematical rounding and range checking, but directly assigns the data types (C-style direct conversion).

Probably the only useful screnario is the conversion int/uint if negative data should be displayed as the last half of the dynamic range, or vice versa. (e.g. 0xFFFFFFFF <-> -1).

Detailed Description

template<class RAW_DATA_TYPE, class COOCKED_DATA_TYPE>
class RegisterTypeAbstractorRawImpl< RAW_DATA_TYPE, COOCKED_DATA_TYPE >

This is a double templated accessor abstractor with raw and coocked data type.

It is based upon the single-typed RegisterTypeAbstractorImpl with the raw type as user type. For those functions which do coocked values, it does the corresponding conversion.

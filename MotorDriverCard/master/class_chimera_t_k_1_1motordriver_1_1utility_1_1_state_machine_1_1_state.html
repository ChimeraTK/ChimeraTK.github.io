Detailed Description

Class describing state of the StateMachine.

This object holds the state name and a TransitionTable describing transitions from an instance to any target states.

This function will append a new state the the TransitionTable of this state.

Parameters
event- Event causing the transition
target- Pointer to the target state
entryCallback- A callback function that will be executed when entering the target state
internalCallback- An optional internal callback that can be used to implement internal events. To achieve this, the callback function must check the condition for a state transition and might then call StateMachine::performTransition. This callback is performed on each call to StateMachine::getCurrentState.

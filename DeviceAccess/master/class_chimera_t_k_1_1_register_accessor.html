TemplateUserTypeMap<AccessorHandler> ChimeraTK::RegisterAccessor::_convertingAccessorHandlers
mutableprotected

The converting accessors used under the hood.

They are not initialised in the constructor but only when first used to safe memory. Usually you will not use read or write of all user data types. Thus the variable is mutable to allow initialisation in the const read and write function.

Detailed Description

MicroDAQ module for logging data to HDF5 files.

This can be usefull in enviromenents where no sufficient logging of data is possible through the control system. Any ChimeraTK::Module can act as a data source. Which variables should be logged can be selected through EntityOwner::findTag().
